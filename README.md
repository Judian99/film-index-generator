# 胶片索引图生成器

一个纯前端网页原型，用于把扫描后的单张底片图片生成 135 胶片风格索引图。

## 使用

直接用浏览器打开 `index.html`，导入 JPG / PNG / WebP 扫描件后即可预览并导出。

当前支持：

- 按文件名、拍摄/修改时间排序，或拖拽自定义排序（预览图上直接拖动帧，或在照片列表中拖拽）
- 时间排序优先读取 EXIF 拍摄时间（DateTimeOriginal）
- 分批追加导入、单张移除、一键清空
- 自动应用 EXIF 方向；导入预处理会按当前画幅及半格输入模式统一方向
- 自动等比居中裁切，不拉伸画面；画幅可选（135 全画幅 / 半格 / 6×6 / 6×4.5 / 6×7 / XPan 宽幅）
- 半格支持两种输入：默认“单张裁切”将每个文件作为一个竖向半格，普通行固定 12 张、模拟片头首行 10 张，并保持标准六格 135 条带尺寸
- “单张未裁切”将一个包含两格的横向扫描文件作为一张 3:2 图，不自动拆分，按全幅规则每行可选 4–8 张
- 已导入图片切换半格输入模式时会从稳定原图重新处理，不会累计自动旋转
- 模拟胶片边字（与齿孔分区排布：上带型号字样、下带帧号与帧界标记）
- 内置胶卷型号下拉选择
- 按 135 规格连续排列的模拟齿孔
- 片基光泽、投影、暗角、颗粒与划痕等真实底片质感
- 预览图自由缩放，放大后可拖拽平移查看细节
- PNG / JPG 导出，导出前校验画布尺寸上限
- 1x / 2x / 3x 输出质量选择

## 说明

图片只在浏览器本地读取和绘制，不上传，也不写入额外元数据。边字目前是模拟效果，后续可以在积累真实胶片边字素材后替换为素材驱动的渲染方式。
