评审重点

  1. 曝光窗比例是否符合所选画幅的真实物理尺寸。
  2. 135 的齿孔、边字与侧边距，以及 120 的平切端头是否自然。
  3. 结构层是否准确贴合片基外轮廓和中央曝光区域。
