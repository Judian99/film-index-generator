Review Notes

评审重点

  1. 摄影棚工作台的空间深度、环境光方向与整体暗角是否自然。
  2. 软投影、接触阴影、片基微纹理和边缘反射是否形成真实材质层次。
  3. 不同画幅下，曝光窗、齿孔、边字和所有质感层是否准确贴合。
